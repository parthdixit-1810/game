How to Play Bollywood Buzz

Objective: Be the first to win 3 rounds. Each round is first to 7 points.

Game Flow

  1. Create or Join: Create a game code or join using an existing code
  2. Enter Names: Both players enter their names
  3. Answer Questions: Questions appear one at a time
  4. Score Points: First to answer correctly gets the point
  5. Readiness Check: After each question, click "Ready for Next?" to continue
  6. Round Winner: First to 7 points wins the round and gives loser a dare
  7. Game Winner: First to win 3 rounds wins the game!

Question Categories

  • Film Trivia: Questions about Bollywood films, actors, and directors
  • Song Lyrics: Complete famous Bollywood song lyrics

Scoring

Each correct answer = 1 point

First to 7 points wins the round

Loser performs a dare

Tips

  • Both players can answer each question
  • First correct answer wins the point
  • If both answer wrong, no one gets the point
  • You have 15 seconds to answer each question
  • Click "Ready for Next?" after you've seen the result
